// wykonaj kod jeśli true
}

Porównania

  • >   lewa strona większa od prawej
  • <   < /span>lewa strona mniejsza od prawej
  • >=  lewa strona większa / równa od prawej
  • <=  < /span>lewa strona mniejsza / równa od prawej
  • ==  obie strony równe
  • === obie strony równe (sprawdza typeof)
  • !=  obie strony różne
  • !== obie strony różne (sprawdza typeof)

M06L08: CSS: grid [17:22]

CSS: Grid

GRID

Świat dwuwymiarowy

Po co ten grid?

Potrzeba stworzenia prostego systemu do budowania layoutów (tabele, floaty, div-oza itp. nie spełniały oczekiwań i były kiepską praktyką)

Przyjazny zapis i możliwość łatwej edycji w kontekście responsywności

Standard dopasowany do aktualnego stanu stron internetowych

źródło: https://webkit.org/blog/7434/css-grid-layout-a-new-layout-module-for-the-web/

Grid container

display:grid

grid-template-columns:<wartości po spacji>

grid-template-rows:<wartości po spacji>

nowa jednostka: fr

1fr 1fr 2fr 444

repeat(ileRazy, wartość)

column-gap: <rozmiar odstępu>

row-gap: <rozmiar odstępu>

minmax(minWartość, maxWartość)

header - article - aside - footer

grid-template-areas:

  • "header header"
  • "article aside"
  • "footer footer"

Grid item

grid-area:

  • <nazwa obszaru>
  • <row-start>
  • <col-start>
  • <row-end>
  • <col-end>

źródło: https://webkit.org/blog/7434/css-grid-layout-a-new-layout-module-for-the-web/

grid-row-start: numer / [nazwa] / span
grid-row-end: numer / [nazwa] / span
grid-column-start: numer / [nazwa] / span
grid-column-end: numer / [nazwa] / span

grid-row: numer / [nazwa] / span
grid-column: numer / [nazwa] / span

z-index

Grid to nie tetris - wszystko zamyka się w regularnych prostokątach

Grid nie zastępuje flexboxa*
*super się uzupełniają
